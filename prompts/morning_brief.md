# Morning Brief Prompt

Generate Steven's morning brief. Be direct and specific.

CRITICAL RULES:
- The REAL ACTIVITY DATA block above this prompt contains actual numbers from Google Sheets. Use those numbers exactly — do not invent or estimate.
- If activity counts are all zero, say so honestly ("No activities logged yesterday").
- If no KPI data is present, omit the KPI section entirely.
- Only report tasks that are real and verifiable from conversation history or memory.
- One insight or suggestion must be genuinely useful, not filler.

Format:

☀️ Good morning Steven.

**Today is [DATE].**

**YESTERDAY'S ACTIVITY:**
[Use the REAL ACTIVITY DATA block if present. Report actual counts: districts researched, emails drafted, calls logged. If all zeros: "Nothing logged yesterday — fresh start."]

**KPI PROGRESS (yesterday vs targets):**
[Only include if REAL ACTIVITY DATA block shows goals. Format: "Calls: X/10 | Districts: X/2 | Emails: X/5". Omit if no goal data.]

**QUEUED / IN PROGRESS:**
[List only real tasks from memory or prior conversation. If nothing: "Nothing queued — ready for your direction."]

**CARRIED OVER FROM YESTERDAY:**
[Only if memory shows unfinished tasks. If nothing: omit this section entirely.]

**NEEDS YOUR INPUT:**
[Questions or decisions that require Steven before Scout can proceed. If none: omit.]

**TODAY'S INSIGHT:**
[One specific, actionable observation about the pipeline, a target district, a market trend, or an outreach improvement. Make it genuinely useful.]

Keep the entire brief under 200 words.
